# 🎬 Hybrid Movie Recommendation System

## Project Overview

This project presents a **Hybrid Movie Recommendation System** developed as part of an MSc Data Science Capstone Project.

The system combines:

* Rule-Based Filtering
* Natural Language Processing (NLP)
* TF-IDF Vectorization
* Cosine Similarity
* Hybrid Recommendation Ranking

to provide personalized movie recommendations based on user preferences and textual descriptions.

## Project Objectives

The primary objective of this project is to develop an intelligent movie recommendation system capable of:

* Filtering movies using user preferences.
* Understanding textual movie descriptions.
* Generating relevant recommendations using machine learning techniques.
* Combining rule-based and content-based recommendation approaches.

---

## Dataset

This project uses the MovieLens dataset.

Files used:

* movies.csv
* ratings.csv
* tags.csv

The dataset provides:

* Movie titles
* Genres
* User ratings
* User-generated tags

---

## System Architecture

```text
User Input
      ↓
Rule-Based Filtering
(Genre + Rating + Year)
      ↓
TF-IDF Processing
      ↓
Cosine Similarity
      ↓
Hybrid Ranking
      ↓
Movie Recommendations
```

## Project Phases

### Phase 1: Data Collection and Preprocessing

Tasks completed:

* Loaded MovieLens dataset.
* Merged movies and ratings datasets.
* Calculated average movie ratings.
* Extracted release years from movie titles.
* Cleaned and prepared data for analysis.

Technologies:

* Python
* Pandas

---

### Phase 2: Rule-Based Recommendation System

Implemented:

* Genre Filtering
* Rating Filtering
* Release Year Filtering

Example:

```python
recommend_movies(
    genre="Comedy",
    min_rating=4.0,
    min_year=2000
)
```

---

### Phase 3: NLP-Based Recommendation

Implemented:

* Text preprocessing
* TF-IDF Vectorization
* Content representation using genres and user tags
* Cosine Similarity calculation

Technologies:

* Scikit-learn
* TF-IDF
* Cosine Similarity

---

### Phase 4: Hybrid Recommendation System

Integrated:

* Rule-Based Filtering
* NLP Similarity Ranking

Hybrid recommendations are generated by combining metadata filtering and textual similarity analysis.

---

### Phase 5: Testing and Evaluation

Evaluation performed using:

* Multiple natural-language queries
* Similarity score analysis
* Recommendation ranking analysis

Example test queries:

* family animation
* space adventure
* romantic drama
* action hero
* comedy movie
